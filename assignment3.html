Rerun anscombe01.R

# Load Anscombe's data
data(anscombe)

# Create four model objects
lm1 <- lm(y1 ~ x1, data=anscombe)
lm2 <- lm(y2 ~ x2, data=anscombe)
lm3 <- lm(y3 ~ x3, data=anscombe)
lm4 <- lm(y4 ~ x4, data=anscombe)

# Preparing for the plots
op <- par(mfrow = c(2, 2), mar = 0.1 + c(4, 4, 1, 1), oma = c(0, 0, 2, 0))

ff <- y ~ x
mods <- list(lm1, lm2, lm3, lm4)  # Store models in a list

# Plot charts using a for loop
for (i in 1:4) {
  ff[2:3] <- lapply(paste0(c("y", "x"), i), as.name)
  plot(ff, data = anscombe, col = "red", pch = 21, bg = "orange", cex = 1.2,
       xlim = c(3, 19), ylim = c(3, 13))
  
  # Extract intercept and slope from each model and add regression line
  abline(coef(mods[[i]]), col = "blue")
}

# Add title
mtext("Anscombe's 4 Regression data sets", outer = TRUE, cex = 1.5)

# Reset plotting parameters
par(op)

Interpretation of the regression plots

All four regressions show the relationship between x and y: they all have the same slope (same \(\beta\) coefficients), but they differ in their true functional relationships.

  • \(x_1\) and \(y_1\) illustrate the best fitted line, minimizing the squared distance.

  • \(x_2\) and \(y_2\) are better captured in quadratic function rather than common linear assumption.

  • \(x_3\) and \(y_3\) illustrate the effect of one outlier point.

  • \(x_4\) and \(y_4\) illustrate the outlier effect when \(x_4=19\) (or \(20\)) and \(x_4\) does not seem to have any variation at all. Yet we still have a linear relationship.

Fine-tuning charts with R basic packages

We just try to change some values in the plotting, e.g., pch for shapes, family (font), etc.

# Scatterplot

# Data points for x, y1, and y2
x <- c(0.5, 2, 4, 8, 12, 16)
y1 <- c(1, 1.3, 1.9, 3.4, 3.9, 4.8)
y2 <- c(4, .8, .5, .45, .4, .3)

# Set up plot
plot.new()
plot.window(range(x), c(0, 6))

# Custom colors
col_y1 <- "darkblue"  # Non-default color for line 1
col_y2 <- "darkgreen" # Non-default color for line 2

# Plot the lines with custom colors
lines(x, y1, col=col_y1, lwd=2) 
lines(x, y2, col=col_y2, lwd=2)

# Custom plotting characters (pch)
points(x, y1, pch=17, col=col_y1, cex=2) # Triangles for y1
points(x, y2, pch=15, col=col_y2, cex=2) # Squares for y2

# Set axis colors and font to serif
par(col="gray50", fg="gray50", col.axis="gray50", family="serif")

# Add axes
axis(1, at=seq(0, 16, 4)) 
axis(2, at=seq(0, 6, 2)) 
axis(4, at=seq(0, 6, 2)) 
box(bty="u")

# Add custom axis labels
mtext("Travel Time (s)", side=1, line=2, cex=0.8) 
mtext("Responses per Travel", side=2, line=2, las=0, cex=0.8) 
mtext("Responses per Second", side=4, line=2, las=0, cex=0.8)

# Add text with serif font
text(4, 5, "Bird 131", col="black", family="serif")

Fine-tuning charts with tidyverse

We can use packages such as tidyverse for customizing the plots:

library(tidyverse)

# Data points for x, y1, y2
data <- tibble(
  x = c(0.5, 2, 4, 8, 12, 16),
  y1 = c(1, 1.3, 1.9, 3.4, 3.9, 4.8),
  y2 = c(4, 0.8, 0.5, 0.45, 0.4, 0.3)
)

# Plot with ggplot2
ggplot(data) +
  # Line for y1 and y2 with custom colors
  geom_line(aes(x = x, y = y1), color = "darkblue", size = 1.2) +
  geom_line(aes(x = x, y = y2), color = "darkgreen", size = 1.2) +
  
  # Points with custom shapes and colors
  geom_point(aes(x = x, y = y1), shape = 17, color = "darkblue", size = 4) +  # Triangles for y1
  geom_point(aes(x = x, y = y2), shape = 15, color = "darkgreen", size = 4) + # Squares for y2

  # Labels
  labs(x = "Travel Time (s)", y = "Responses per Travel") +

  # Add text annotation
  annotate("text", x = 4, y = 5, label = "Bird 131", family = "serif", size = 5) +

  # Custom theme with serif font
  theme_minimal(base_family = "serif") +
  
  # Customize axis ticks and labels
  theme(
    axis.title = element_text(size = 12, color = "gray50"),
    axis.text = element_text(color = "gray50"),
    axis.line = element_line(color = "gray50"),
    panel.grid.minor = element_blank(),
    panel.grid.major = element_line(color = "gray80")
  )
